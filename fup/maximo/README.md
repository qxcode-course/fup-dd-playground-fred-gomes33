# Valor máximo entre dois números

![Imagem eh Positivo](../../.tko/cache/e2bc1de5ac270eaa50476053dc6bb1f9ff79f7ba/base/maximo/assets/cover.jpg)

## Ação

Implemente um programa que recebe dois números inteiros e imprime o maior.

### Entrada

- Dois inteiros

### Saída

- Imprima o maior número

## Testes

```py
>>>>>>>> INSERT 0
4
9
======== EXPECT
9
<<<<<<<< FINISH
```

```py
>>>>>>>> INSERT 1
56
7
======== EXPECT
56
<<<<<<<< FINISH
```

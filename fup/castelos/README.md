# Castelo de Cubos

![_](../../.tko/cache/e2bc1de5ac270eaa50476053dc6bb1f9ff79f7ba/base/castelos/assets/cover.jpg)

Vamos montar castelos utilizando a seguinte lógica.

1² = 1 ![_](../../.tko/cache/e2bc1de5ac270eaa50476053dc6bb1f9ff79f7ba/base/castelos/assets/quadrado1.png)

---

2² = 4 ![_](../../.tko/cache/e2bc1de5ac270eaa50476053dc6bb1f9ff79f7ba/base/castelos/assets/quadrado2.png)

---
3² = 9 ![_](../../.tko/cache/e2bc1de5ac270eaa50476053dc6bb1f9ff79f7ba/base/castelos/assets/quadrado9.png)

---
4² = 16 ![_](../../.tko/cache/e2bc1de5ac270eaa50476053dc6bb1f9ff79f7ba/base/castelos/assets/quadrado16.png)

---
5² = 25 ![_](../../.tko/cache/e2bc1de5ac270eaa50476053dc6bb1f9ff79f7ba/base/castelos/assets/quadrado25.png)

Esses números, 1, 2, 4, 9, 16, 25, etc, eram chamados números quadrados ou quadrados perfeitos.

Dado um inteiro lido do usuário, determine, sem uso de operadores reais (como raiz quadrada, por exemplo), se ele é um quadrado perfeito, imprimindo "sim" se ele for quadrado perfeito e "nao", caso contrário.

### Entrada

- Um número inteiro.

### Saída

- sim ou nao
